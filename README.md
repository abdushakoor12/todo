# To-Do App with Flutter && GetX

To Do App with all CRUD operation

## Tech

- Flutter (UI)
- GetX for State Management, Navigation
- GetStorage for shared Preferences
- Notification with localzone
- MVC pattern
- Sqflite for local storage
